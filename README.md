

# Memscan 内存检索工具

使用Go编写的Windows 应急响应轻量工具，快速扫描全进程内存中的恶意域名 / IP / 特征码，自动关联进程活跃 TCP 外联，一分钟定位恶意进程。

## ✅ 核心功能

- 全进程可读 / 可执行内存区域扫描
- 支持正则匹配、大小写区分
- 自动关联进程 TCP 活跃外联信息
- PID 合并 / 平铺双模式结果展示
- 结果实时过滤与 TXT 报告导出
- 单 EXE 无依赖，隔离网即插即用

## 🚀 快速使用

1. 下载 Releases 页面的 `Memscan.exe`
2. **右键 → 以管理员身份运行**（必须）
3. 输入检索关键词，点击「立即扫描」
4. 扫描完成后可过滤进程、切换展示模式、导出报告

## 🛠️ 自行编译

```
# 安装Fyne命令行工具
go install fyne.io/fyne/v2/cmd/fyne@latest

# 一键打包（自动嵌入图标，生成单EXE）
fyne package -os windows -icon ico.png
```
